总结2017，展望2018

18年已经过了几天了，可是我的17年总结一直不知从何下手。
回想起自己碌碌无为的17年，在看看17年初的计划发现自己完成的并不是很满意。

那天看到一个段子，我觉得这不就在说我吗，它是这样说的

  • 2017年只剩一个月了，你的年度计划完成的怎么样呢，我呀，凡是花钱的计划基本落实，凡是赚钱的计划基本没戏。
  • 给大家分享一些我2018年的小目标，别过的像2017年就好了。
  • 2018年新年计划，搞定2017年那些原定于2016年完成的安排，不为别的，只为兑现2015年实现完成2014年计划的承诺。

我想说我也是这样认为的O(∩_∩)O！

17年总结

17年的经历了太多的坎坷，虽说收货不多，但是肯定还是有的。我就来简单的总结一下，就不一一叙述了。

  • 17年学了小半年RN。
  • 学习了很多关于测量方面的知识
  • 一个人开发了3个android项目
  • 经历了搬家两次，搬公司两次就是搬过去又回来
  • 看了两本技术书–太少了
  • 搭建了自己的博客
  • 参加了几场技术大会
  • 买了MacBook Pro

17年前半年我过的很快，快的让我没有感觉，可能我在前半年的收获很小，没有可能，就是很小，的确很小。最令我印象深刻的是后半年，一个人开发的这几个项目，有一个我感到很骄傲，从中也学到了很多新东西，不断的优化，精益求精。这个项目为了赶进度，我那几个月经常回去也在加班开发到12点，也遇到了很多不会的问题，感谢群里的好朋友帮我解答，谢谢你们。在这个项目小有成就的时候，领导看到了给我涨了薪。所以说付出总会有收获。

关于17年，不想啰嗦的太多了，因为有太多的遗憾，我终将这些遗憾化成18年一年前进奋斗的动力。
接下来我们就直奔主题吧，展望2018，让18年不平凡，别过的像17年那样。

跨年感悟

年级大了，面对各种压力，就会变的焦虑。我们这代人的焦虑应该怎么解决。面对这样的焦虑我们更重要的抓住更多的机会。跨年的时候我看了罗振宇演讲的时间的朋友，罗胖逐步看清当下机会所在。这些机会，是只能发生在我们身边的“中国式机会”。他也提出了6个脑洞：

动车组脑洞 / 热带雨林脑洞 / 比特化脑洞 /拔河脑洞 / 终点站脑洞 / 枢纽脑洞

我们要怎么抓住中国式机会呢，那就是人生算法。算法这个词我并不陌生，写代码有时候就会用到。可是人生算法是什么呢？罗胖他是这样说的：

在过去一年的最后一天，我们看到了“中国式机会”，但是——能抓住机会的终究是少数人。
我们还需要一套人生算法。
人生算法不是一把解决所有问题，需要我们具备概率的思维模式，通过不断反馈取得越来越大的成就。
通过算法的迭代，成功的概率也在不断提高。这就是“人生算法”的力量。任何好事，都只是“可能性”，需要通过努力、实践，还有算法，去逐渐逼近它。

总结为一句话：人生算法就是你面对世界不断重复的最基本的套路，找到它，重复它，加强它。你抓住中国式机会，就是更大概率的事件。

18年计划

  • 读5本技术书籍
  • 深入JAVA
  • 学数据结构和算法
  • 学会前端和后台
  • 制定英语学习计划
  • 写一些有质量的博客
  • 锻炼身体
  • 学会理财
  • 平常心

这只是18年的一个大致的目标，具体的计划下来还要好好斟酌。一切的计划还是要按照人生算法规则，不断的重复它，加强它。也就是我们平常所说的坚持，能坚持下来的都是赢家。

我想对你说

  • 身体健康最重要
  • 要努力学英语
  • 看更多的书
  • 做事有计划有目标，不断总结自己
  • 要有一颗感恩的心
  • 还有很多,我就不唠叨了

End

2018年已经开始，大家已经分头前进，各自启动自己的人生算法。算法的力量，会给我们足够的回报。(๑•̀ㅂ•́)و✧加油！！！
